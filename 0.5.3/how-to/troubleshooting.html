Minecraft Server wont start

If you see this error:

docker.errors.DockerException: Error while fetching server API version: ('Connection aborted.', FileNotFoundError(2, 'No such file or directory'))

This implies that the docker engine is not running. Use the start menu to launch Docker Desktop and retry once it is running.

Problems with Player Connection

If you see a message like this:

WARNING:        failed to give signs to player, player <player> left

This means MCIWB could not find your player. It is worth checking that you have your player name correct.
